# superpowers ディレクトリについて

本ディレクトリは Toique の **過去フェーズ実装計画書 / 設計書のアーカイブ** です。

- `plans/` … フェーズごとの実装計画書 (Phase 1 / 2a / 2b / 3 等)
- `specs/` … プロジェクト全体の設計書

## 留意事項

- 各ファイルは **執筆時点の計画** を記録したもので、最終的な実装内容や現行のリポジトリ状態とは必ずしも一致しません。
- 計画書中に「認証なし」「平文保存」など現状の弱点を示唆する記述があった場合でも、それは **当時のフェーズ計画段階の前提** です。最新の実装ステータスは [`README.md`](../../README.md) と [`docs/ROADMAP.md`](../ROADMAP.md) を参照してください。
- 計画書は OSS の開発ジャーニーの記録としてリポジトリに残しています。引用や流用は MIT License の範囲で自由に行えます。
